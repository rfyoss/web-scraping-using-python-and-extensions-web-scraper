Proyek ini bertujuan untuk melakukan proses web scraping data e-commerce serta melakukan pembersihan dan pengolahan data menggunakan Python. Notebook ini menunjukkan alur lengkap mulai dari membaca data hasil scraping, memperbaiki struktur dataset, hingga melakukan scraping langsung dari website menggunakan library Python seperti BeautifulSoup dan Playwright.
Proyek ini cocok sebagai latihan dasar dalam memahami proses data collection dan data structuring sebelum masuk ke tahap analisis data lebih lanjut.
Pada tahap awal, dataset hasil scraping (file CSV) dibaca menggunakan library pandas. Setelah data dimuat, dilakukan eksplorasi awal seperti menampilkan beberapa baris pertama dan terakhir untuk memahami struktur data serta memeriksa nama kolom.
Selanjutnya dilakukan proses perapihan dan penggantian nama kolom agar lebih terstruktur dan mudah dipahami. Setelah struktur data diperbaiki, dataset disimpan kembali ke dalam file baru sebagai bentuk dokumentasi dan memastikan data telah bersih serta siap digunakan untuk analisis lebih lanjut.
